# BreakOpen

A breakout clone that I'm using to re-teach myself some game dev coding, while also learning Rust.

## Prerequisites

- [Install Rust](https://www.rust-lang.org/en-US/)

## Use

- Navigate to the root directory and `cargo build && cargo run`
